HOST GUIDE

Tempo Host Instruction Book

From first login to a full multi-round show — prep, save, print, play.

Written for teen readers: short steps, plain words.

1. What you’re doing

You’re hosting music bingo: songs play, players mark squares, someone wins.

Tempo pulls tracks from Spotify and/or YouTube Music, helps you plan rounds (like chapters), and lets you prep, save, and print cards before the crowd shows up.

TabUse it for…
ManagerPlaylists, rounds, music login, round prep
GameLive show: play snippets, host controls

2. Before you open Tempo

  • Use Chrome or Edge on a laptop if you can.
  • Have Spotify and/or YouTube Music ready — match whatever you’ll play from.
  • Know how you sign in as host (only hosts drive the game).

3. First-time in the room

A. Open your room

Join your Tempo room as host. Players join separately — they don’t pick playlists.

B. Connection

  1. Open Connection (Spotify / device / YouTube — labels vary).
  2. Connect Spotify if you use Spotify playlists.
  3. Connect YouTube Music if you use YouTube-only playlists.
No connection = Tempo can’t see your playlists. Fix this before anything else.

C. Pick the speaker

Choose where audio goes (speaker, mixer, Bluetooth). Test volume once.

4. Playlists & pool modes

Five playlists (5×15 style)

  • You pick five playlists.
  • Each one lines up with a column on the big call board when the room is in that layout.
  • Great when each list has a theme (90s, rock, one artist, etc.).

One huge playlist (1×75 style)

  • One list with lots of songs turns into one shared pool.
  • Simpler vibe, fewer columns to think about.

If Tempo warns you about “not enough songs,” add tracks or pick different lists — don’t ignore it.

5. Rounds = chapters of your night

Rounds split the event (opening, themed game, finale). Each round usually has:

  • A name
  • Playlists tied to that round
  • A status (planned → playing → done)

You move rounds forward when your schedule says so — Tempo doesn’t rush you.

6. Load for prep (practice without an audience)

You do not need to “start the live game” just to get organized.

Load for prep:

  • Puts that round’s playlists into the mix
  • Syncs your bingo pattern controls with that round
  • Lets you rehearse while it’s still quiet
Use this before doors open: load Round 2, tweak pattern, hit Save round, print PDFs — no crowd required.

7. Save round (freeze your rehearsal)

Save round stores a snapshot of that round after the mix rules run (trimming, layout, etc.).

  • Show night can match what you rehearsed.
  • If you change playlists a lot, save again so the snapshot stays true.

8. Finalize mix (lock the bingo pool)

Finalize builds the official song pool for cards and display.

  • Required before printing PDFs and solid digital cards.
  • If it fails: playlists didn’t load, rate limits, or connection glitch — refresh library, wait a minute, retry.

9. Printable PDF cards

  • Cards per PDF = how many unique printed sheets (often 30).
  • Round & event management → Print PDF and Game tab → Download PDF use the same rules on the server — same random logic, same bingo shape. Only the PDF subtitle may show the round name.
  • You still must finalize first — same rule everywhere.
  • Set card count on the Manager tab or inside the round popup — they stay matched.

10. Round & event management (the popup)

Open Round prep & PDF from Manager.

Inside you’ll see overview stats, card count, and per-round actions:

  • Load for prep
  • Save round
  • Print PDF
  • Start when it applies

Use this when you’re thinking about the whole event, not only “right now on stage.”

11. Running a multi-round live game

  1. Manager: playlists on rounds look correct.
  2. Prep: load round → finalize if needed → save round → print if you need paper.
  3. Game tab: Start Game when the room is ready.
  4. Music plays as snippets; titles show on the public display.
  5. Players mark cards → bingos get checked.
  6. Next round: finish or advance the round, load for prep for the next playlists, finalize/save if your workflow needs it, then continue.

If the projector columns look wrong after big edits, you probably need to finalize again so the locked pool matches what everyone sees.

12. Quick fixes

Empty playlistsReconnect Spotify/YouTube; refresh library
Finalize failedWait out rate limits; fewer giant playlist pulls at once
Weird printed cardsFinalize after your last playlist change
YouTube vs Spotify confusionUse the service you actually connected

13. Night-before checklist

  • ☐ Host login works
  • ☐ Spotify or YouTube connected
  • ☐ Speaker device picked
  • ☐ Every round has playlists
  • ☐ Load-for-prep tested on rounds you care about
  • ☐ Save round after a happy rehearsal
  • ☐ PDF printed if you need paper
  • ☐ Pattern + free-space double-checked
  • ☐ Public display bookmark / casting ready

Prep slow, party loud. You’ve got this.
